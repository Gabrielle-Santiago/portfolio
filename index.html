E-commerce de Roupas

E-commerce de Roupas (Gestão de Venda e Compra)

Este projeto simula uma plataforma de comércio eletrônico completa, onde possui dois fluxos distintos e seguros para Vendedores e Clientes.

O sistema gerencia o acesso e a segurança de dados (como métodos de pagamento) utilizando Tokens (Token e Refresh Token) e um sistema de Roles para garantir que apenas Vendedores autorizados possam gerenciar produtos (cadastrar, editar ou deletar). Atualmente está em desenvolvimento, o foco principal é na construção de uma arquitetura robusta para uma experiência de compra e venda confiável.

Delivery Backend

Sistema de Pedidos de Delivery

Plataforma de delivery completa onde os clientes podem navegar, fazer pedidos e simular pagamentos. O sistema garante que o administrador da loja seja notificado por e-mail no mesmo instante em que um pedido é finalizado, permitindo uma resposta rápida e eficiente. O projeto foca em organizar o acesso e proteger os dados, criando uma experiência de compra e gestão de produtos fluida e confiável.

Catálogo de Jogos Backend

GameCatalog: Catálogo de Jogos

Aplicação central para orquestrar e disponibilizar dados de jogos, notas e gêneros, consumindo e processando informações da API externa OpenCritic. O projeto é construído em uma arquitetura modular (DTO, Service, Controller e Client), que facilita a manutenção e o desenvolvimento de novas funcionalidades. O foco é na integração eficiente e na organização limpa do código, garantindo que o usuário tenha acesso rápido e detalhado a informações específicas de jogos por meio de rotas bem definidas, incluindo busca personalizada.

Agendamento Automatizado

Agenda Pro: Agendamento Automatizado e Chat em Tempo Real

Sistema focado em automatizar o agendamento de consultas médicas, respeitando regras complexas de horário e disponibilidade. O projeto possui diferenciais técnicos de alta complexidade, como a integração direta com o Google Calendar API para sincronização automática de compromissos e a implementação de Chat em Tempo Real entre usuários via WebSocket.
